Heart Trail Animation ❤️

This is a beginner mini project built using HTML, CSS, and JavaScript.
It creates a heart trail animation that follows the mouse movement on the screen.

Features:
- Heart trail effect on mouse movement
- Random heart sizes
- Smooth upward animation
- Auto removal of elements after animation

Technologies Used:
- HTML
- CSS
- JavaScript

How to Run:
- Open index.html in any modern browser
- Move the mouse to see the animation
